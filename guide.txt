# Start : 4 Tir 1401

1 - Install Laravel or get from Github
  * https://github.com/HadBoard/FFPack

2 - install SQL Server adn SSMS

3 - install SQL Server driver for PHP (ts version for php 7.4)
  * https://docs.microsoft.com/en-us/iis/application-frameworks/install-and-configure-php-on-iis/install-the-sql-server-driver-for-php
  * extension=php_pdo_sqlsrv_74_ts_x64.dll
  * extension=php_sqlsrv_74_ts_x64.dll

4 - Set DB : SQL Server
  * DB_CONNECTION=sqlsrv
  * DB_HOST="DESKTOP-xxxxxxx"
  * DB_PORT=null
  * DB_DATABASE=FFPack
  * DB_USERNAME=sa
  * DB_PASSWORD=password

5 - Run Migration : php artisan migrate
